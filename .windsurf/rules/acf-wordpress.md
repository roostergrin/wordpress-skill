# ACF WordPress Skills

This project manages ACF (Advanced Custom Fields) for a headless WordPress site.
Use `skills/*.md` as the authoritative workflow docs and prefer `wp-acf` as the command surface.

- **Building a page from a design/screenshot** — read `skills/workflow.md` first.
- **Verifying setup before schema or content work** — read `skills/wp-acf-preflight.md`.
- **Creating or modifying ACF field groups** — read `skills/acf-schema-edit.md`.
- **Deploying schema to WordPress** — read `skills/acf-schema-deploy.md`.
- **Reading or updating page/post content** — read `skills/wp-acf-content-api.md`.
- **Path configuration** — read `skills/config.md`.
